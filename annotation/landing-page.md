## 🚀 Fase 2 — Landing Page: HTML e CSS Puro

### 🎯 Objetivo
Criar a interface do usuário — um formulário funcional e limpo.

### 🛠️ Tecnologias
- HTML5  
- CSS3 (sem frameworks)  
- JavaScript vanilla (apenas para validação básica no front)

### 📌 Tarefas Práticas

#### 1. Estrutura HTML
Criar um `index.html` contendo um formulário com:

- Campo de texto para digitar o código de barras  
- Botão de envio  
- Campo de e-mail de destino  

#### 2. Estilização (CSS)
- Layout centralizado  
- Design responsivo para dispositivos mobile  

#### 3. Validação (JavaScript)
- Validar se o campo de código de barras possui **47 ou 48 caracteres** antes do envio  

#### 4. Testes
- Testar no navegador localmente utilizando um arquivo estático
